join      Commandes Internes Tcl

NOM

join - Crée une chaîne en joignant ensemble les éléments d'une liste

SYNTAXE

join list ?joinString?

DESCRIPTION

L'argument list doit être une liste Tcl valide. Cette commande retourne la chaîne formée en joignant tous les éléments de list ensemble avec joinString separant chaque paire d'éléments adjacents. L'argument joinString est par défaut un caractère espace
