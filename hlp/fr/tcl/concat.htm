concat      Commandes Internes Tcl

NOM

concat – concatène des listes

SYNTAXE

concat ?arg arg ...?

DESCRIPTION

Cette commande traite chaque argument comme une liste et les concatène dans une simple liste. Elle élimine aussi les espaces de début et de fin dans les arg's et ajoute un simple separateur espace entre les arg's. Elle permet quelconque nombre d'arguments. Par exemple, la commande
concat a b {c d e} {f {g h}}
renverra
a b c d e f {g h}
comme résultat.
Si aucun args n'est fourni, le résultat est une chaîne vide.
