cd       Commandes Internes Tcl

NOM

cd - Change le répertoire de travail

SYNTAXE

cd ?dirName?

DESCRIPTION

Change le répertoire de travail courant en dirName, ou en répertoire home (défini par la variable d'environnement HOME) si dirName n'est pas donné. Retoune une chaîne vide.
