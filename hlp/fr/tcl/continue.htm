continue      Commandes Internes Tcl

NOM

continue - Passe à l'itération suivante d'une boucle

SYNTAXE

continue

DESCRIPTION

Cette commande est typiquement appelée dans le corps d'une commande de boucle telle que for ou foreach ou while. Elle renvoie un code TCL_CONTINUE, qui provoque une exception continue. L'exception provoque l'arrêt du script courant, et la boucle continue avec l'itération suivante. Les exceptions sont aussi gérées dans d'autres situations, telles que la commande catch et les scripts les plus externes des corps de fonction.
