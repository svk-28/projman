append      Commandes Internes Tcl

NOM

append - Ajoute à une variable

SYNTAXE

append varName ?value value value ...?

DESCRIPTION

Ajoute tous les arguments value à la valeur courante de la variable varName.  Si varName n'existe pas, il lui est donné une valeur égale à la concaténation de tous les arguments value. Cette commande fournit un moyen efficace de construire des variables longues par incrémentation. Par exemple, “append a $b” est beaucoup plus efficace que “set a $a$b” si $a est long.
