linsert      Commandes Internes Tcl

NOM

linsert – Insère des éléments dans une liste

SYNTAXE

linsert list index element ?element element ...?

DESCRIPTION

Cette commande produit une nouvelle liste à partir de  list en insérant tous les arguments element juste avant l'élément index de list. Chaque argument element deviendra un élément séparé de la nouvelle liste. Si index est inférieur ou égal à zéro, alors les nouveaux éléments sont insérés au début de la liste. Si index a la valeur end, ou s'il est supérieur ou égal au nombre d'éléments dans la liste, alors les nouveaux éléments sont ajoutés à la liste. end-integer se réfère au dernier élément dans la liste moins l'offset entier spécifié

Dernière révision: 8.2
