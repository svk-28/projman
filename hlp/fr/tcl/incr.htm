incr      Commandes Internes Tcl

NOM
incr - Incrémente la valeur d'une variable

SYNTAXE

incr varName ?increment?

DESCRIPTION

Incrémente la valeur stockée dans la variable dont le nom est varName. La valeur de la variable doit être un entier. Si increment est fourni alors sa valeur (qui doit être un entier) est ajouté à valeur de la variable varName;  autrement 1 est ajouté à varName. La nouvelle valeur est stockée comme chaîne décimale dans la variable varName et également renvoyée comme résultat
