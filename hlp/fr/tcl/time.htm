time      Commandes Internes Tcl

NOM
time - chronomètre l'exécution d'un script

SYNTAXE

time script ?count?

DESCRIPTION

Cette commande appelera l'interpréteur Tcl count fois à évaluer script (ou une fois si count n'est pas spécifié) Elle retournera ensuite une chaîne de la forme
503 microseconds per iteration
ce qui indique le temps moyen requis par itération, en microsecondes. Time est mesuré en temps écoulé, pas en temps CPU
