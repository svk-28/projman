break      Commandes Internes Tcl

NOM

break – Sort d'une commande de boucle

SYNTAXE

break

DESCRIPTION

Cette commande est habituellement appelée dans le corps d'une commande de boucle tel que for ou foreach ou while. Elle renvoie un code TCL_BREAK, qui cause une exception break. L'exception cause l'arrêt du script courant à l'intérieur de la commande de boucle, qui cesse alors son exécution et retourne normalement. Les exceptions break sont aussi gérées dans quelques autres cas, tels que la commande catch, les bindings d'évènement Tk, et les scripts externes des corps de fonction.
