regsub

Команда выполняет подстановки, используя регулярные выражения.

СИНТАКСИС

      regsub ?switches? exp string subSpec varName

ОПИСАНИЕ

Команда сравнивает регулярное выражение exp и строку string и копирует string в переменную, заданную именем varName. Если совпадение найдено, то при копировании часть строки string, соответствующая exp, замещается на subSpec. Если subSpec содержит один из символов & or
